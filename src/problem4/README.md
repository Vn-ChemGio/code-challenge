# TypeScript Function Implementations Task

## Overview

This project provides **3 unique TypeScript implementations** of the same function.

### Requirements

* The function accepts:

  * `n`: **any integer**
* The function must return a valid result for all supported inputs.
* You may assume the result will always be less than `Number.MAX_SAFE_INTEGER`.

## Evaluation Criteria

Each implementation should:

1. Be **functionally correct**.
2. Use a **different approach / algorithm**.
3. Include a **complexity / efficiency analysis**.
4. Include a comparison in the **JSDoc comments**.


## Notes

* The actual comparison and trade-offs between implementations are documented in the **JSDoc comments inside the code**.
* Focus on readability, correctness, and performance trade-offs.
